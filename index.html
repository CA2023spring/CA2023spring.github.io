Computer architecture is the foundation of computer science and technology. This course aims to establish the design, evaluation, and optimization of computer architecture methods and technologies through the use of quantitative analysis techniques. Using these methods and technologies, one can effectively evaluate the performance of existing computer systems, improve the existing system designs, and propose new architectures.

This course has 60 lecture hours and 40 lab hours. The topics include the fundamentals of quantitative analysis of computer systems, reviews of instruction set architecture design and basic pipeline technology, memory system optimization, instruction level parallelism, data level parallelism and task level parallelism.
